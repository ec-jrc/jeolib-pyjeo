# pyJEO

pyJEO is a library for image processing for geospatial data implemented in 
JRC Ispra. 

# License

# Dependencies

 * jiplib
 * numpy

# Install

From the directory of the repository, run:
```
sudo python setup.py install
```

# Test the installation

From the directory of the repository, run:
```
python -m unittest tests
```
